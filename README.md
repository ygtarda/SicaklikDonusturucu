# Sıcaklık Dönüştürücü Uygulaması

Bu basit Java Swing uygulaması, Celsius derecesini Fahrenheit derecesine dönüştürmek için tasarlanmıştır.

## Özellikler

* **Sıcaklık Dönüşümü:** Kullanıcının girdiği Celsius değerini Fahrenheit'a dönüştürür.
* **Modern Arayüz:** FlatLaf teması sayesinde modern ve koyu bir arayüze sahiptir.
* **Özel Buton:** Mavi renkte ve oval kenarlı özel bir dönüşüm butonu içerir.
* **Başlangıç Ekranı:** Uygulama açılırken kısa bir yükleme ekranı (splash screen) gösterir.

## Nasıl Çalıştırılır

1.  **Gereksinimler:**
    * Java Development Kit (JDK) 11 veya üzeri
    * IntelliJ IDEA gibi bir Java IDE (Gradle desteği ile)

2.  **Projenin Açılması:**
    * Projenizi (sicaklik-donusturucu) IntelliJ IDEA'da açın.
    * IDE'nin Gradle dosyalarını senkronize etmesini bekleyin.

3.  **Uygulamanın Başlatılması:**
    * `src/main/java/com/sicaklik/app/App.java` dosyasını bulun.
    * `main` metodunu içeren bu dosyayı çalıştırın.

## Kullanım

1.  Uygulama açıldıktan sonra "Celsius Değeri:" alanına bir sayı girin.
2.  "Dönüştür (Celsius → Fahrenheit)" butonuna tıklayın.
3.  Sonuç, "Sonuç:" etiketinin yanında görüntülenecektir. Geçerli bir sayı girilmezse bir hata mesajı alırsınız.
